Introduction to Elice Programming

환영합니다! 간략하게 elice 홈페이지에서 프로그래밍을 어떻게 하는지 알아보겠습니다.

실행
먼저 상위에 위치한 실행 버튼을 클릭해 보기 바랍니다. 실행 버튼을 클릭하시면, 가운데 화면에 있는 Python 코드가 실행됩니다. (코드는 자동 저장됩니다!)

제출
주어진 과제를 완료한 뒤에는 소스 코드의 제출 버튼을 눌러 주시면 됩니다. 제출된 소스 코드는 채점되어 오른쪽 화면에 결과가 나타납니다.

과제
이번 예제에서는 따로 소스 코드를 수정할 필요 없이, 제출 버튼을 눌러서 코드를 제출 및 검증해 보시기 바랍니다.

중요: Elice 웹 프로그래밍에서는 Python 3 버전을 사용합니다. Python 2 버전에 익숙하신 수강생께서는 파이썬 2.7 vs 파이썬 3 (한국어) 혹은 Cheat Sheet: Writing Python 2-3 compatible code (영어) 를 참고하시면 도움이 되실 것입니다. 가장 주의해야 할 두 가지는 다음과 같습니다.

print: Python 3에서는 출력할 문자열에 괄호를 필요로 합니다.
input: Python 2에서의 raw_input 과는 달리 3에서는 input 을 사용합니다.
Python은 오픈 소스 프로그래밍 언어로, 보기 좋으면서도 읽기 쉬운 코드를 만들기 위해 제작되었습니다. Python은 인터프리트 언어로 프로그래머가 컴퓨터에 소스 코드를 주면 바로 실행이 가능한 언어입니다. Python은 확장성이 크며 많은 커뮤니티에서 이용되고 있는데 특히 scipy, numpy, matplotlib 와 같은 강력한 패키지를 통해 scientific computing을 쉽게 할 수 있습니다.
